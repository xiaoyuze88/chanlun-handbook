### 任务1：从 graphify 生成概念-课文索引与超边关系

读取以下文件：
- `graphify-out/GRAPH_REPORT.md`
- `graphify-out/.graphify_chunk_01.json` ~ `.graphify_chunk_08.json`（读所有存在的）

**处理步骤**：

**Step A：建立全局 id→label 映射表**
合并所有 chunk JSON 中的 `nodes` 数组，每个 node 有 `id` 和 `label` 字段。
建立映射表：`{id: label}`，供后续翻译超边节点使用。

**Step B：统计神节点**
合并所有 chunk JSON 中的 `edges` 数组，统计每个节点 ID 的出边+入边总数（degree）。
取 degree 最高的前10个节点，通过 id→label 映射表翻译为可读名称。
同时从每个节点的 `source_file` 字段，汇总该概念出现在哪些课文中。

**Step C：提取超边关系**
合并所有 chunk JSON 中的 `hyperedges` 数组。每个超边有：
- `id`：超边标识
- `label`：超边名称（可读）
- `nodes`：节点 ID 列表（需用 id→label 映射表翻译为 label）
- `confidence`：字符串，`EXTRACTED` 或 `INFERRED`
- `confidence_score`：浮点数（如 0.85、0.95）

将节点 ID 翻译为 label，写入超边关系表。

**Step D：提取概念-课文完整索引**
对每个 node，从 `source_file` 字段提取来源课文文件名。
根据以下关键词规则判断节点所属大类（P1~P8）：
- P1：哲学、心理、理性、当下、人性、贪嗔痴、修炼
- P2：分型、笔、线段、特征序列
- P3：中枢、走势中枢、ZG、ZD、GG、DD
- P4：走势类型、走势分解、级别、同级别分解
- P5：背驰、MACD、趋势力度
- P6：买点、卖点、买卖点
- P7：均线吻、区间套、操作级别、资金管理、止损
- P8：案例、图解、牛市、庄家、大资金
- 不确定的标注 `?`

**Step E：提取社区聚类**
从 `graphify-out/GRAPH_REPORT.md` 的 Communities 章节，提取每个 community 的名称和包含的主要节点（课文）列表。

输出 `systematic/00-CONCEPT-MAP.md`，格式如下：
